# C Programming Practice

## Overview
This repository is a personal collection of C programming exercises and examples. It is organized by topic to help review core concepts, strengthen problem-solving skills, and build confidence in C programming.

## Topics
- Variables — Fundamental practice with variable declaration and usage.
- Constants — Working with fixed values in programs.
- Data Types — Understanding basic and derived data types.
- Operators — Practice with arithmetic, relational, and logical operators.
- Booleans — Simple boolean logic and condition handling.
- Flow Control — Control structures for program decision-making.
- Switch — Branching with switch statements.
- Break — Using break statements in loops and switch cases.
- Continue — Skipping iterations with continue.
- For Loop — Repetition using for loops.
- While Loop — Repetition using while loops.
- Function — Writing and using reusable functions.
- Array — Working with collections of values.
- String — Handling and manipulating strings.
- Pointer — Understanding memory addresses and pointer operations.
- Advanced Pointer — More complex pointer-based practice.
- Memory Allocate — Dynamic memory allocation.
- Recursion — Solving problems through recursive functions.
- User Defined Data Types — Structs, enums, and unions.
- File Handling — Reading and writing files.
- Embedded C — Basic embedded C examples.
- Examples — Small demonstration programs.
- Overflow — Practice with numeric overflow scenarios.
- DSA — Data structures and algorithms practice.

## Installation
To run the programs, you need a C compiler installed on your machine.

And for debugging, install a debugger also.

### Windows (MSYS2 UCRT64)
1. Install MSYS2.
2. Open the UCRT64 terminal.
3. Verify the compiler:

```bash
gcc --version
```

## How to Run
Compile a C file with GCC:

```bash
gcc -o output.exe path/to/file.c
```

Run the compiled program:

```bash
./output.exe
```

Example:

```bash
C:/msys64/ucrt64/bin/gcc.exe -o bai1.exe array/bai1.c
./bai1.exe
```
